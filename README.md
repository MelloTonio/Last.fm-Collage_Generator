# Last.fm Collage Generator

Monta uma "Grid" de albuns a partir da escolha do usuário por meio de requisições ao servidor do last.fm

### Prerequisites

Bibliotecas necessárias para rodar:
* Axios
* Feito com auxilio do Browsify

```npm init ```  
 ```npm install axios```
